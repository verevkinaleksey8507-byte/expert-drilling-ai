# Настольные версии

Windows и macOS используют оболочку Tauri и тот же пакет `offline-dist`, который собирается из общего расчётного ядра `lib/calculations.ts`.

Порядок выпуска:

1. `node scripts/build-offline.mjs` — автономный интерфейс.
2. Сборка Tauri на целевой операционной системе.
3. Автоматический прогон контрольных примеров.
4. Подпись установщика сертификатом разработчика.
5. Проверка установки, обновления и сохранности проектов.

Основные расчёты не используют сеть. CSP настольной оболочки запрещает сетевые подключения.
